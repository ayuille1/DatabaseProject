Folder contents:

-templates
	-contains all of the .html files used for the interface of the project
	-base.html: the basic template for the website, used by all other pages in the folder
	-home.html: the home of the website, the first page you see when opening the site
	-crop_insert.html: page used to insert data to the crop table
	-field-insert.html: page used to insert data to the field table
	-greenhouse-insert.html: page used to insert data to the greenhouse table
	-sales-insert.html: page used to insert data to the sales table
	-display.html: page template used to display tables

-crop.sql: generated data for the crop table

-farm.sql: generated data for the farm table

-fertilizer.sql: generated data for the fertilizer table

-field.sql: generated data for the field table

-greenhouse.sql: generated data for the greenhouse table

-sales.sql: generateed data for the sales table

-seed.sql: generated data for the seed table

-crop_data.sql: SQL script that creates the tables and schema to hold the data above

-crop_data.py: the python script that activates the application and provides functionality to it
	-to use: run the script, in the terminal copy and paste the IP address that appears in the browser, use the navbar to navigate the website

-cropData_ERDiagram.drawio: Draw.io file used to create the ER Diagram describing the schema of crop_data

-cropData_ERDiagram.png: Image of the Draw.io file

-DS_ProjectReport_AustinYuille.docx: Report for the project
